Trocifreni parni brojevi

Motivacija

Trocifreni brojevi (od 100 do 999) su posebno zanimljivi jer predstavljaju sve moguće kombinacije tri cifre. U ovom zadatku tražimo samo one koji su parni i koji se nalaze u zadatom intervalu [a,b].

Naivno rešenje

Mogli bismo da napišemo sve parne trocifrene brojeve ručno: 100, 102, 104, ... 998. Ali to je predugačko i nepraktično, naročito ako radimo sa intervalima. Zato nam je potreban sistematski pristup.

Naivni pristupi često pomognu da shvatimo obrazac, ali nas takođe navode ka efikasnijem rešenju.

Analiza problema

Prvi korak ka uspešnoj realizaciji jeste jasno razumevanje svih zahteva, a zatim planiranje kako ih sprovesti na najefikasniji mogući način.
Da bismo rešili zadatak:

  • Prvo moramo proveriti da li je trenutni broj trocifren (100 ≤ n ≤ 999).
  • Drugo, da li je broj paran (n % 2 == 0).
  • Treće, da li se nalazi unutar zadatog intervala [a,b].

Ako ispunjava sva tri uslova, ispisujemo ga, pa prelazimo na sledeći broj.

Pravilan pristup

Osnovni slučaj: ako trenutni broj pređe vrednost b, zaustavljamo se. Induktivni korak: ako je broj trocifren i paran, ispisujemo ga; zatim prelazimo na sledeći broj n+1. Tako dobijamo algoritam koji prolazi kroz sve brojeve u intervalu.

Primeri i testovi

void IspisiParneTrocifrene(int a, int b) { if (a > b) return; if (a >= 100 && a <= 999 && a % 2 == 0) Console.WriteLine(a); IspisiParneTrocifrene(a + 1, b); } static void Main() { int a = int.Parse(Console.ReadLine()); int b = int.Parse(Console.ReadLine()); IspisiParneTrocifrene(a, b); }
Ovde koristimo induktivno razmišljanje: ako znamo da ispišemo rešenje od a+1 do b, onda dodajemo a ako zadovoljava uslove.
