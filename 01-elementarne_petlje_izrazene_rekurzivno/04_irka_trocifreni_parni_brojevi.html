Trocifreni parni brojevi

Motivacija

Trocifreni brojevi (od 100 do 999) su posebni jer su to sve kombinacije tri cifre. U ovom zadatku želimo da pronađemo samo one koji su parni i koji se nalaze u zadatom intervalu [a,b]. Na primer, ako je interval [95, 110], ispisaćemo: 100, 102, 104, 106, 108, 110.

Naivno rešenje

Mogli bismo da napišemo sve parne trocifrene brojeve ručno: 100, 102, 104, ... 998. Ali to bi bilo dosadno i nepraktično, posebno kada radimo sa promenljivim intervalima. Potrebno nam je pametnije, sistematsko rešenje.

Naivni pristupi su korisni da prepoznamo obrazac, ali obično nisu efikasni kada se zadatak proširi.

Analiza problema

Pre nego što pišemo kod, razjasnimo uslove:

  • Broj mora biti trocifren: 100 ≤ n ≤ 999.
  • Broj mora biti paran: n % 2 == 0.
  • Broj mora biti u zadatom intervalu [a,b].

Ako broj ispunjava sva tri uslova, ispisujemo ga i prelazimo na sledeći broj.

Induktivni pristup

U rekurziji uvek imamo dva ključna dela:

  • Osnovni slučaj: Ako trenutni broj pređe gornju granicu b, zaustavljamo se.
  • Induktivni korak: Ako trenutni broj ispunjava uslove (trocifren i paran), ispisujemo ga, a zatim pozivamo postupak za a+1.

Primer koda

void IspisiParneTrocifrene(int a, int b) { if (a > b) return; // osnovni slučaj if (a >= 100 && a <= 999 && a % 2 == 0) Console.WriteLine(a); // ispisujemo samo ako je trocifren i paran IspisiParneTrocifrene(a + 1, b); // prelazimo na sledeći broj } static void Main() { int a = int.Parse(Console.ReadLine()); int b = int.Parse(Console.ReadLine()); IspisiParneTrocifrene(a, b); }
Ovaj kod pokazuje induktivno razmišljanje: ako znamo da ispišemo sve parne trocifrene brojeve od a+1 do b, onda samo proverimo i dodamo a ako ispunjava uslove.

Pitanja za samoevaluaciju

  • Koji uslovi moraju da budu ispunjeni da bi broj bio ispisan?
  • Po čemu se ovaj algoritam razlikuje od zadatka „Brojevi od a do b“?
  • Kako bismo promenili kod da ispisuje neparne trocifrene brojeve?

Ako ne znate odakle da krenete, razmislite:

  • Koji je prvi trocifreni broj? A poslednji?
  • Kada broj treba da bude ispisan?
  • Šta se dešava kada pređemo gornju granicu b?
