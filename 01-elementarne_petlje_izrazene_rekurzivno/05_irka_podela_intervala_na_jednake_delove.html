Podela intervala na jednake delove

Motivacija

Zamislite da imate lenjir dužine od a do b i želite da ga podelite na jednake delove. Da li biste crtali sve tačkice unapred ili biste krenuli od prve, pa zatim dodavali jednu po jednu? Upravo ovakvo razmišljanje nam pomaže da problem rešimo sistematski – induktivno.

Naivno rešenje

Naivno rešenje bi bilo da unapred izračunamo i ispišemo svaku vrednost. Međutim, ako želimo da podelimo interval na mnogo delova, ovo postaje nepraktično. Potrebno nam je algoritamsko rešenje koje radi za bilo koji unos.

static void Intervali(int n, double a, double b) { double dx = (b - a) / (n - 1); for (int i = 0; i < n; i++) { Console.WriteLine((a + i * dx).ToString("0.00000")); } } // Primer poziva: Intervali(5, -1, 1);
Naivno rešenje nam daje rezultat, ali nije induktivno konstruisano.

Analiza problema

Da bismo ispisali n brojeva ravnomerno raspoređenih od a do b, potrebno je:

  • izračunati razmak dx = (b - a) / (n - 1)
  • ispisati prvi broj a
  • zatim ispisati sledeći broj a + dx, pa onda a + 2dx, sve do b

Induktivno posmatrano: ako znamo kako da ispišemo prvi broj, onda znamo i kako da ispišemo sledećih n-1 brojeva – jednostavno dodajemo dx.

Induktivni pristup

Osnovni slučaj: ako je n = 0, prekidamo. Induktivni korak: ispišemo trenutni broj a, pa zatim pozivamo funkciju za brojeva počevši od a + dx.

Primeri i testovi

static void RavnomernoRasporedjeniBrojevi(int n, double a, double dx) { if (n > 0) { Console.WriteLine(a.ToString("0.00000")); RavnomernoRasporedjeniBrojevi(n - 1, a + dx, dx); } } static void Main() { int n = int.Parse(Console.ReadLine()); double a = double.Parse(Console.ReadLine()); double b = double.Parse(Console.ReadLine()); double dx = (b - a) / (n - 1); RavnomernoRasporedjeniBrojevi(n, a, dx); }
Ovde jasno vidimo induktivnu konstrukciju: osnovni slučaj je prekid, a induktivni korak dodaje sledeću tačku u intervalu.
