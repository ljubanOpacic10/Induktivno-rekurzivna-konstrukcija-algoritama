Faktorijel

Motivacija

Faktorijel je jedan od najpoznatijih matematičkih izraza i često se pojavljuje u kombinatorici, statistici i programiranju. Označava proizvod svih prirodnih brojeva od 1 do n i zapisuje se kao n!. Na primer: 5! = 1 × 2 × 3 × 4 × 5 = 120.

Zanimljivo je da se faktorijel pojavljuje svuda — od računanja mogućih kombinacija karata, do broja načina da se ljudi rasporede u red. Iako se lako može izračunati pomoću petlje, faktorijel je sjajan primer kako induktivno razmišljanje i rekurzija mogu elegantno rešiti problem korak po korak.

Naivno rešenje

Najjednostavniji način bio bi da koristimo petlju i redom množimo sve brojeve od 1 do n. Ovaj pristup funkcioniše, ali ne pokazuje zašto to radi, niti kako se rešenje prirodno gradi. Rekurzivni (induktivni) pristup nam pomaže da bolje razumemo kako se svaki korak nadovezuje na prethodni.

Analiza problema

Faktorijel broja n možemo izraziti induktivno:

\( f(n) = n \times f(n - 1) \)

Dakle, da bismo izračunali n!, dovoljno je da znamo (n−1)! i da ga pomnožimo sa n. Kada stignemo do 1, više nemamo šta da množimo — to je osnovni slučaj.

Osnovni slučaj: Ako je n = 0 ili n = 1, tada je f(n) = 1.

Induktivni korak: Ako znamo faktorijel broja (n−1), tada je f(n) = n × f(n−1).

Pravilan pristup

Funkciju možemo zapisati rekurzivno. U svakom pozivu funkcija računa faktorijel manjeg broja dok ne dođe do osnovnog slučaja. Postoji i repno–rekurzivna verzija koja je efikasnija, jer se ne mora „pamti“ više poziva.

int Faktorijel(int i, int n, int f) { if (i > n) return f; // osnovni slučaj: nema više brojeva return Faktorijel(i + 1, n, f * i); // induktivni korak } int Faktorijel(int n) { return Faktorijel(1, n, 1); }

i — trenutni broj koji trenutno množimo (počinje od 1).

n — broj do kog računamo faktorijel.

f — privremeni rezultat koji sadrži proizvod svih prethodnih brojeva.

Praćenje izvršavanja

Za n = 3, tok funkcije izgleda ovako:

  • Faktorijel(1, 3, 1) → prelazimo na sledeći broj (1 × 1 = 1)
  • Faktorijel(2, 3, 1) → sada množimo 1 × 2 = 2
  • Faktorijel(3, 3, 2) → sada množimo 2 × 3 = 6
  • Faktorijel(4, 3, 6) → i > n → vraćamo 6

Dakle, funkcija „prenosi“ delimičan rezultat kroz parametar f, sve dok ne pređe granicu n. Na kraju dobijamo n! = 6.

  • Zašto kažemo da je ova funkcija repno–rekurzivna?
  • Kako bi izgledala obična rekurzivna verzija funkcije za faktorijel?
  • Šta bi se desilo ako zaboravimo da definišemo osnovni slučaj?

Zadaci za vežbu

1. Napiši funkciju koja računa dvostruki faktorijel broja n (n!! = n × (n−2) × (n−4) × ...).

2. Napravi funkciju koja računa zbir svih faktorijela brojeva od 1 do n.

3. Napiši jednostavniju verziju funkcije bez pomoćnih parametara (i, f).

4. Objasni kako se induktivno razmišljanje koristi u svakoj od ovih funkcija.

Pomoć – Faktorijel

Induktivno razmišljanje uvek kreće od najjednostavnijeg slučaja (n = 1), pa zatim gradi rešenje korak po korak. Za faktorijel, to znači da znamo da je 1! = 1, a zatim svaki sledeći rezultat dobijamo množenjem prethodnog sa sledećim brojem.

Na primer, ako računamo 4!:

  • 1! = 1
  • 2! = 2 × 1 = 2
  • 3! = 3 × 2 = 6
  • 4! = 4 × 6 = 24

Dakle, svaka nova vrednost koristi rezultat prethodne — baš kao što rekurzija koristi rešenje manjeg problema za sledeći korak.
