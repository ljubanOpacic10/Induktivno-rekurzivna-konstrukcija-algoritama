Faktorijel

Motivacija

Faktorijel je jedan od najpoznatijih matematičkih izraza koji se često pojavljuje u kombinatorici, statistici i programiranju. Označava proizvod svih prirodnih brojeva od 1 do n, i zapisuje se kao n!. Na primer, 5! = 1 × 2 × 3 × 4 × 5 = 120. Iako se može lako izračunati pomoću petlje, faktorijel je odličan primer za induktivno razmišljanje i primenu rekurzije.

Naivno rešenje

Jedan jednostavan način je korišćenje petlje koja računa proizvod svih brojeva od 1 do n. Iako ovakav pristup funkcioniše, ne pokazuje kako se rešenje gradi korak po korak, niti razvija sposobnost induktivnog razmišljanja.

Analiza problema

Faktorijel broja n može se prirodno izraziti induktivno: f(n) = n × f(n − 1), dok je osnovni slučaj f(0) = 1. To znači da ako znamo rezultat za n−1, lako možemo izračunati rezultat za n.

Osnovni slučaj — kada je n = 0 ili n = 1, rezultat je 1.

Induktivni korak — ako znamo faktorijel broja n−1, onda je faktorijel broja n jednak n × (n−1)!.

Pravilan pristup

Funkciju možemo zapisati rekurzivno. Svaki rekurzivni poziv računa manji problem (faktorijel od n−1), dok se osnovni slučaj završava kad n dostigne 1. U ovom primeru koristićemo repno-rekurzivni pristup, koji se po strukturi ponaša kao i iteracija.

int Faktorijel(int i, int n, int f) { if (i > n) return f; return Faktorijel(i + 1, n, f * i); } int Faktorijel(int n) { return Faktorijel(1, n, 1); } // Ulaz: ceo broj n // Izlaz: faktorijel broja n (n!)

Rekurzivna funkcija Faktorijel(i, n, f) ima tri parametra, a svaki od njih ima svoju jasnu ulogu:

i — trenutni broj koji obrađujemo (počinje od 1 i povećava se pri svakom rekurzivnom pozivu).

n — krajnja vrednost do koje računamo (npr. ako želimo 5!, tada je n = 5).

f — trenutni proizvod svih brojeva do i−1, odnosno „privremeni rezultat“ koji se prosleđuje sledećem pozivu.

Tok izvršavanja možemo pratiti na primeru n = 3:

Faktorijel(1, 3, 1) => poziv za broj 1, trenutno f = 1
Faktorijel(2, 3, 1) => sledeći korak, sada množimo f × i = 1 × 1
Faktorijel(3, 3, 2) => i = 3, množimo f × i = 2 × 3
Faktorijel(4, 3, 6) => sada je i > n, pa se vraćamo rezultat = 6

Dakle, funkcija rekurzivno gradi rezultat, prenoseći kroz parametar f delimičan proizvod iz prethodnih koraka. Kada broj i pređe n, više nema novih brojeva za množenje — i tada se dobija konačan rezultat, koji je n!.

Zadaci za vežbu

1. Napiši funkciju koja računa dvostruki faktorijel broja n (n!!).
2. Napiši funkciju koja vraća zbir svih faktorijela brojeva od 1 do n.
3. Napravi rekurzivnu funkciju koja računa n! bez pomoćnih parametara (i, f).
4. Objasni kako se u svakoj varijanti koristi induktivno razmišljanje.

Pomoć

Podseti se da induktivno razmišljanje polazi od najjednostavnijeg slučaja i gradi ka složenijem. Za faktorijel, osnovni slučaj je n = 1. Zatim u svakom sledećem koraku rešenje „nasleđuje“ prethodno i samo se pomnoži novim brojem n. To je upravo princip induktivne konstrukcije algoritma.
