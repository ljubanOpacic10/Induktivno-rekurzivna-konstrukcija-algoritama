Stepen broja

Motivacija

Računanje stepena broja je jedan od najčešćih zadataka u matematici i programiranju. Na primer, 2³ znači da broj 2 treba pomnožiti sam sa sobom tri puta: 2 × 2 × 2 = 8. Umesto da odmah koristimo ugrađene funkcije, hajde da analiziramo kako se ova operacija može konstruisati induktivno.

Naivno rešenje

Jedan jednostavan način je korišćenje petlje koja ponavlja množenje x sa samim sobom n puta. Ovakav algoritam radi, ali ne pokazuje način razmišljanja – kako se rezultat gradi korak po korak. Da bismo razumeli suštinu, pokušaćemo da formulišemo algoritam koji sledi induktivnu logiku.

Analiza problema

Posmatrajmo stepen broja xⁿ. Ako želimo da ga izračunamo induktivno, možemo posmatrati odnos između xⁿ i xⁿ⁻¹ (xⁿ = x × xⁿ⁻¹).
Drugim rečima, ako znamo vrednost xⁿ⁻¹, samo je potrebno da rezultat pomnožimo sa još jednim x.

Osnovni slučaj — kada je n = 0, rezultat je 1 (jer je svaki broj na nultom stepenu jednak 1).

Induktivni korak — ako znamo xⁿ⁻¹, tada je xⁿ = x × xⁿ⁻¹.

Pravilan pristup

Koristićemo repno-rekurzivnu funkciju koja postepeno gradi rezultat. Funkcija prima tri parametra:

x — broj koji stepenujemo,

s — trenutna vrednost proizvoda (početno je 1),

n — broj preostalih množenja koja treba obaviti.

Kada n dostigne 0, rezultat je gotov i funkcija vraća konačnu vrednost.

double Stepen(double x, double s, int n) { if (n == 0) return s; return Stepen(x, s * x, n - 1); } double Stepen(double x, int n) { return Stepen(x, 1.0, n); } // Ulaz: broj x i stepen n // Izlaz: vrednost xⁿ

Tok izvršavanja možemo pratiti na primeru x = 2 i n = 3:

Stepen(2, 1, 3) => početni poziv, s = 1, još treba da pomnožimo tri puta
Stepen(2, 2, 2) => s = 1 × 2 = 2, preostalo množenje: 2 puta
Stepen(2, 4, 1) => s = 2 × 2 = 4, preostalo množenje: 1 put
Stepen(2, 8, 0) => s = 4 × 2 = 8, n = 0, kraj rekurzije

Dakle, funkcija se završava kada n = 0 i vraća konačan rezultat s = 8. To znači da je 2³ = 8.

Zadaci za vežbu

1. Napiši funkciju koja računa xⁿ i za slučaj kada je n negativan (x⁻ⁿ = 1 / xⁿ).
2. Napiši funkciju koja računa kvadrat broja (x²) bez korišćenja rekurzije.
3. Pokušaj da napišeš funkciju koja koristi samo jednu funkciju (bez pomoćnog parametra s).
4. Objasni gde se u svakom od ovih zadataka koristi induktivni korak.

Pomoć

Induktivno razmišljanje u ovom zadatku podrazumeva da razumemo kako se svaki sledeći rezultat gradi iz prethodnog. Ako znamo xⁿ⁻¹, jednostavno ga množimo sa x da bismo dobili xⁿ. Repno-rekurzivna funkcija samo automatizuje ovaj proces, bez dodatnih računanja nakon poziva funkcije.
