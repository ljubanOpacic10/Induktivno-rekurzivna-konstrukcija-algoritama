Prva negativna temperatura

Motivacija

Na meteorološkoj stanici svakog dana meri se temperatura. Želimo da pronađemo prvu negativnu temperaturu — prvi dan kada je temperatura bila ispod nule. Ovaj zadatak pokazuje kako rekurzija može da se koristi za pretragu elemenata, a ne samo za računanje zbirnih vrednosti.

Naivno rešenje

Jedan jednostavan način je da koristimo petlju i proveravamo redom svaku temperaturu dok ne naiđemo na prvu negativnu. Takvo rešenje radi, ali induktivni pristup nam omogućava da isto postignemo bez eksplicitne petlje — kroz logiku koja sama „pamti“ gde se trenutno nalazi.

Analiza problema

Funkcija prima dve vrednosti: trenutnu poziciju i i ukupan broj elemenata n. Ako je i > n, nema više elemenata i funkcija vraća -1. U suprotnom, učitavamo sledeću temperaturu:

• Ako je negativna, odmah vraćamo trenutnu poziciju (to je prva negativna vrednost).

• Ako nije negativna, prelazimo na sledeću poziciju, tj. pozivamo funkciju za i + 1.

Pravilan pristup

Koristimo repno-rekurzivnu funkciju koja korak po korak proverava svaku temperaturu.

int PrviNegativan(int i, int n) { if (i > n) return -1; int x = int.Parse(Console.ReadLine()); return x < 0 ? i : PrviNegativan(i + 1, n); } int PrviNegativan(int n) { return PrviNegativan(1, n); } // Ulaz: broj n i n temperatura // Izlaz: pozicija prve negativne temperature ili -1 ako nema negativnih

Objašnjenje toka izvršavanja

Tok izvršavanja možemo pratiti na primeru:
n = 5
Uneti brojevi: 3, 8, -2, 1, 0

PrviNegativan(1, 5) => učitavamo 3 (nije negativna) → prelazimo na i = 2
PrviNegativan(2, 5) => učitavamo 8 (nije negativna) → prelazimo na i = 3
PrviNegativan(3, 5) => učitavamo -2 (negativna!) → vraćamo poziciju 3

Dakle, funkcija se završava kada pronađe prvi negativan broj i vraća njegovu poziciju — u ovom slučaju 3.

2. Napiši funkciju koja pronalazi poslednju negativnu temperaturu.
3. Napiši funkciju koja vraća koliko ukupno ima negativnih temperatura.
4. Objasni kako se u svim ovim zadacima koristi induktivni pristup.

Pomoć

Induktivno razmišljanje u ovom zadatku znači da svaki korak rešava manji deo problema: ako trenutna temperatura nije negativna, problem se svodi na preostale temperature. Kada dođemo do prve negativne vrednosti ili do kraja liste, imamo rešenje.
