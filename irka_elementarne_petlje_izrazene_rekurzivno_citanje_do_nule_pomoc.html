Pomoć - Čitanje do nule

Setite se da ovde nije poznat broj elemenata unapred. Algoritam se završava tek kada unesemo nulu.

Osnovni slučaj: ako unesemo 0 → vraćamo 0 i tu se rekurzija zaustavlja. Induktivni korak: ako unesemo broj ≠ 0, dodajemo 1 i pozivamo istu metodu za sledeći unos.

Na primer, za unos: 7, 4, 9, 0 → algoritam radi ovako:

  • Prebroj(7) → 1 + Prebroj(4)
  • Prebroj(4) → 1 + Prebroj(9)
  • Prebroj(9) → 1 + Prebroj(0)
  • Prebroj(0) → 0
Ukupan rezultat = 3.
