Geometrijska serija

Motivacija

Zamislite da imate niz brojeva gde se svaki sledeći dobija množenjem prethodnog sa tri. Ovakav niz nazivamo geometrijska serija. Takve serije se često koriste za modelovanje rasta (na primer, bakterija koje se utrostručuju) ili u zadacima sa brojevima koji rastu eksponencijalno.

Naivno rešenje

Jedan od naivnih pokušaja bio bi da prolazimo kroz sve brojeve u intervalu [a, b] i proveravamo da li pripadaju seriji. Međutim, to je neefikasno, jer proveravamo i mnogo nepotrebnih brojeva.

Analiza problema

Ono što primećujemo jeste da brojevi u geometrijskoj seriji nastaju jednostavnim pravilom: prvi broj je a, a svaki sledeći se dobija množenjem prethodnog sa 3. Dakle, dovoljno je da krenemo od a i uvećavamo dok ne pređemo gornju granicu b.

Pravilan pristup

Umesto da prolazimo kroz ceo interval, možemo koristiti induktivni korak: ako znamo da je prethodni broj u seriji validan, onda je naredni samo 3 puta veći. Na ovaj način gradimo rešenje korak po korak.

static void GeometrijskaSerija(int a, int b) { int x = a; while (x <= b) { Console.WriteLine(x); x *= 3; } } // Primer poziva u Main metodi: static void Main() { int a = int.Parse(Console.ReadLine()); int b = int.Parse(Console.ReadLine()); GeometrijskaSerija(a, b); }
